# bandirma-deprem-analizi
Bu proje, Balıkesir ili Bandırma ilçesindeki resmi deprem toplanma alanlarının kentsel erişilebilirliğini sokak ağı üzerinden 5 ve 10 dakikalık yaya yürüyüş sürelerine (Isochrone) göre modeller. Proje, metrik doğruluk için TUREF/TM30 (EPSG:5254) projeksiyon sistemiyle geliştirilmiştir.
